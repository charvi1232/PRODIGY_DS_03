PRODIGY_DS_03

Decision Tree Classifier using the Bank Marketing Dataset

Objective

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

Steps Performed

- Loaded the Bank Marketing dataset.
- Encoded categorical variables using LabelEncoder.
- Split the dataset into training and testing sets.
- Trained a Decision Tree Classifier.
- Evaluated the model using accuracy, classification report, and confusion matrix.
- Visualized the Decision Tree.

Results

- Successfully trained a Decision Tree model.
- Evaluated the model's performance.
- Generated a visualization of the Decision Tree.
